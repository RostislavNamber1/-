Пошаговая инструкция для git команд

Создание нового репозитория

  1. git init

    // Производим инициализацию репозитория

  2. git status

    // Проверка на добавления репозитория

  3. git add .

    // Предворительное Добавление всех изменений в репозиторий

  4. git commit -m ""

    // Окончательное добавление в репозиторий и коминт

  5. git remote add origin https://.........

    // Подключение к репозиторию

  6. git push -u origin master

    // Выгрузка всех обновлений для репозитория

Подключение к готовому репозиторию

  1. git clone https://.........

    // Подключение к репозиторию

  2. git pull

    // Обнавление файлов репозитория скачивает только только измененные и недостающие файлы
