Простое горизонтальное меню с отступами

  • Home
  • Home1
  • Home2
  • Home3
            
      <nav>
       <ul>
        <li><a href="#">Home</a></li>
        <li><a href="#">Home1</a></li>
        <li><a href="#">Home2</a></li>
        <li><a href="#">Home3</a></li>
       </ul>
      </nav>
            
          
          
            li, a{
              all: unset; /*reset для всех браузеров*/
            }
            /* меню */
            nav{
              width: 100%;
              min-height: 10px;
            }
            /* настройки для не нумеровоного списка */
            nav ul{
              display: flex;
              flex-flow: row wrap;
              justify-content: center;
            }
            /* настройки для самих ссылок */
            nav ul li a{
              min-width: 1px;
              min-height: 1px;
              color: #ffffff;
              padding: 0 10px 0 10px;
              background-color: rgb(0, 0, 0);
              font-size: 150%;
            }
            /*Добавляем отступы к элиментам*/
            nav li {
              border-right: 10px solid #f20000;
            }
            /*Убираем с последнего ненужный отступ*/
            nav li:last-child {
              border-right: none;
            }
